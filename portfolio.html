HikeIt! UT - Social Hiking App

A group collaboration project that allows users to look up trails in Utah, save them to their own hikes list, join other existing hikes other users have created, and even an open chat room to coordinate hikes. My contributions to the final project were UX/UI and design-based, including the logo.

Nintendo Memory Game

A simple, yet fun memory game using React deployed on Heroku, using Nintendo characters from the game Super Smash Bros. Can you find all of the characters without clicking on the same image twice? It's not as easy as it sounds, because the characters shuffle each time you choose a character.

Crystal Collector jQuery Game

A simple click-based game using jquery for the game logic. The game will generate a random number, all you have to do is click on the gems until their value is equal to the target score. Be careful though! The values of the gems are also random. It's a challenge of luck and memory.
